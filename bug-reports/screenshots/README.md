# Screenshots & Attachments

This folder holds the visual evidence referenced by each bug report (one set per bug, prefixed with
the bug ID), for example:

```
bug-001-cart-total-stale.png
bug-001-network.har
bug-002-negative-total.png
bug-003-search-500.png
```

**Naming convention:** `bug-<id>-<short-description>.<ext>`

In a real project these are also attached directly to the defect ticket (JIRA / Azure DevOps) so the
evidence travels with the bug through its lifecycle. Placeholders are listed here to keep the sample
self-documenting without committing binary screenshots to the repo.
